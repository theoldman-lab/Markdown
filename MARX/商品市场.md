# 商品市场

## 商业资本

这里作出一个比较重要的设计，即商业资本（资本论第三卷）融合到商品市场中。因此，产业资本与商品市场之间的交互就变成了产业资本与商业资本之间的交互：产业资本将产品按照“生产价格”贩卖给商业资本，商业资本使用自己的预付货币资本购买下这些产品后，根据“商业价格”在市场上将这些商品贩卖出去，最后得到一个差价作为它的利润。商业资本按照自己的资本份额在总的资本中获得一个利润分成。关于这个商业利润的问题不再过多论述，参照资本论第三卷的内容。

这里需要指出的是，我们作出了一个比较特别的简化，也就是我们忽略了商业资本对劳动力的雇佣。商业资本会必须付出一个份额，作为它必须承担的商业流通费用，这个费用可以视作商业资本的“不变资本”部分。但是，我们不考虑商业资本雇佣劳动力的部分。因为，第一，商业资本雇佣劳动力在一定程度上可以算进其流通费用的一个技术部分；第二，这个雇佣的劳动力相对于商业资本整体而言是一个很小的部分，而商业资本对劳动力的雇佣相对于产业资本对劳动力的雇佣就更小了。因此，商业资本雇佣劳动力对于系统整体而言都是一个非常小的部分。最后，商业资本不同于产业资本，商业资本的利润不是通过榨取它所雇用的劳动力的剩余价值得到的，而只是通过在产业资本生产出来的剩余价值中分成一个相应于自身资本额的份额得到。这些雇佣劳动力本身不生产价值，只是降低商业流通的费用。商业资本的技术力的进步主要体现在它必须付出的流通费用的降低。

将商业资本集成进商品市场中的设计是一个不很准确的简化，因为这里是直接将社会所有商业资本全部视作一个统一的整体，因此也就是一个超级垄断资本。就这个假设本身来说，他会导致许多谬误，但是，这里为了更加简单地分析产业资本群的发展，将商业资本直接进行一个理想化的简化相对来说是可以忍受的。有了这样一个商业资本的存在，产业资本便从商品流通的重负中解脱出来，会使得他的生产周转更加顺利。这个将商品转化为货币的风险现在就转移到了商业资本的肩上。如果商业资本可以顺利地卖出他的商品，那么商业资本就能够收获一个属于他的利润。反之，如果商业资本不能完成这个任务，那么他就不得不从自己的货币资本中拿出一部分弥补亏空。

关于这个商业利润，它本身按照商业资本自身的资本额乘上一个“平均利润率”得到。至于这个平均利润率，根据资本论第三卷的论述，取决于平均构成产业资本的利润率得到。这里可能有待优化补充。因为此处的商业资本直接被简化为一个整体，因此说明这个商业利润可能会比较抽象。

最后，商业价格，即生产价格加上商业利润之后，应该还有一个份额，被金融资本拿走。（后续会提到，金融资本也被集成进货币市场），因此，此处暂时不做过多的介绍。只是提到生产价格，这里应该和资本论第三卷一样，在加入对商业资本的考虑后，对其定义做一个补充修正。在之前计算平均利润率时，是按照社会总体利润除社会总体产业资本额得到的。但是，现在由于商业资本也加入一个份额到社会总资本中，所以对平均利润率的计算，其分子不变，还是社会全体产业资本生产出来的全部利润（剩余价值），但是分母需要加上社会全体商业资本额。新的生产价格也就成为根据这个新的平均利润率乘产业资本额，再加上生产成本得到。

总之，一句话，虽然真正意义上的利润（剩余价值）只由产业资本生产，但是社会全体产业资本和商业资本都根据自己的资本份额瓜分这个利润。这种瓜分通过资本之间的竞争实际完成。

## 价格

在资本主义经济中，不同生产部门的**资本有机构成**（不变资本c与可变资本v的比例）和**资本周转时间**存在显著差异。根据之前的论述：假设剩余价值率（m'）相同（即剥削程度相同），那么如果商品按价值（c+v+m）出售，不同行业的利润率会因资本有机构成不同而不同。  

如果某些行业利润率长期高于其他行业，资本就会涌入高利润行业，导致供给增加、价格下降，最终利润率下降；反之，低利润行业资本流出，供给减少，价格上升，利润率提高。**因此，竞争使不同行业的利润率趋于平均化**，形成**一般利润率（平均利润率）**。但这一现象似乎与劳动价值论矛盾——如果利润平均化，商品就不再严格按价值出售，而是按**生产价格**（成本价格 + 平均利润）出售。  

```math
\text{平均利润率} \ (p') = \frac{\text{全社会剩余价值总量}}{\text{全社会总资本}} = \frac{\sum m}{\sum (c + v)}\\\\
\text{生产价格} = (c + v) + p' \cdot (c + v) 
```

生产价格取代价值成为市场价格波动的中心。

关于供需问题，首先，马克思区分了**个别价值**和**市场价值**：

- **个别价值**：单个企业在特定生产条件下生产商品所耗费的劳动时间。
- **市场价值**：由社会必要劳动时间决定，通常表现为：**部门内商品的平均价值**；**在平均生产条件下生产的、占市场主要份额的商品的个别价值**。

**市场价值的调节方式**：

- 如果供给与需求平衡，市场价值由**中等生产条件**决定；
- 如果供给严重不足，市场价值由**最差生产条件**决定（因商品稀缺，高价仍能售出），此时，市场价格 > 市场价值；
- 如果供给严重过剩，市场价值由**最优生产条件**决定（因竞争激烈，低价才能售出），此时，市场价格 < 市场价值。

供求关系只能解释市场价格**偏离**市场价值，但不能决定市场价值本身，市场价值的本质仍由社会必要劳动时间决定。

因此，想要完成价格的仿真，我们首先需要找到平均利润率，然后产业资本在出售他的总产品给商业资本时会根据这个利润率计算出总产品价格。商业资本在加上他的商业利润后得到初步的商业价格，最后会根据供需情况调整商业价格。（这里还是没有谈到金融利润的部分）。

那么，如何找到这个平均利润率。如果直接统计全社会剩余价值总量和全社会总资本，那么这是可以做到的。但是，这是不真实的。因为这个平均利润率本身是资本流动和竞争的结果。而这个流动与竞争的标杆是不同资本的不同利润率（商业上实际计算的是年利润率）。我们先假设我们已经有了一个平均利润率（通过直接的统计计算得到），那么接下来考虑这个平均利润率被扰动以不断形成新的平均利润率。

我们在这里假设，商品市场上同类商品的价格都是大致一样的。因此，资本想要获得不同的利润，一方面，取决于资本的产量；另一方面，取决于资本的生产成本。这两方面都和资本的生产力相关。资本之间的生产力固然是不同的。单个资本生产力的发展——偶然的技术进步或者单纯就是对劳动力的加强压榨都会让各个资本之间的发展出现不平衡，从而得到不同的生产成本和产量。对于产业资本来说，他的生产成本是明确的，他制定他的生产价格时，根据先前的平均利润率得到他的利润，再经过商业资本的利润加入以及金融资本的利息加入，得到最终的商品价格（计算供需影响）。商品市场上所有的商品（这里应该是所有售出的商品，没有被售出的商品是无所谓利润的）价格的总和应该等于总商品价值。根据总商品价值减去社会总资本，我们就可以得到全社会总体利润，即全社会剩余价值总量，从而得到新的平均利润率。

这个价格形成的方式确实很不成熟，后续也可以设计其他更好的方案。主要问题在于，一时半会找不到较好的方法实现资本的流入流出。其次，这里的矛盾在于由于我们将商业资本统一为一体，因此商业利润的计算就必须需要一个平均利润率。但是产业资本形成这个平均利润率的过程又不是明确的。所以不得不做出这种粗暴的处理。还有，如果商业资本无法将这些购买来的商品全部出售出去，存货会带来一个单纯的流通费用，其次，商品还会（按量）损耗。再加上供需问题对价格的影响，系统到底能不能实现社会总产品价值等于社会总商品价格和社会总利润等于社会总剩余价值确实存在问题。

这里同类商品的生产价格是不一致的，因为不同资本的生产力不同。商业资本（商品市场）收购这些产品时也就会给出不同的生产价格。但是，同类商品的商业价格却是一致的。因为同一个商业资本收走所有同类商品。虽然他从不同资本购买来这些商品花了不同的生产价格，但是它可以将所有这些同类商品汇总然后根据他购置这些商品所花的全部预付资本计算他所需要加价的商业利润，最后得到这一批商品的统一商业价格。这样就可以避免复杂的价格控制。

最后，供需对价格的影响计算。这主要是由商业资本进行处理。供需情况会刺激产业资本的生产，但是不会直接影响产业资本的生产价格。商业资本会根据他所面临的供需情况适当调整商品价格。比如，当供给不足，商业资本就会在基础的商业价格上，根据情况增加一个额外值。这也就提高了商业资本的利润率。但是，系统中所有的价格都会被支付。所以这个超额商业利润本质上，类似于产业资本得到的超额剩余价值，是对社会其他资本的剩余价值的榨取。反之，如果需求不足，商业资本就不得不降低价格。此时商业利润会降低，甚至商业资本的周转都会受损，那么此时商业资本就不得不额外预付资本弥补亏空。商业资本会将所有产业资本的产品全部收走并且不会有特别的讲价行为，无论供需如何。但是，如果我们在商业资本严重受损的情况下看到，商业资本甚至不足以收走产业资本的全部商品，那么产业资本也就的确无法卖出他的产品。这些产品就不得不闲置下来甚至浪费。这毫无疑问会打击产业资本的周转。至于商业资本具体会如何根据供需来调整它的价格，需要更加细致的研究。

## 流通与再生产

这里主要说的是这样一种流通：不同产业资本之间，相互依赖对方的产品，在市场上购买相应生产资料。在产业资本的定义中，已经提到，同一部类内部，资本之间产品与生产资料的相互依赖关系呈现出一种金字塔形的层级关系，简单表现为产品的不断加工。其次，即便是在同一层级之间也存在相互的依赖关系。接下来作更加详细的描述。

我们假设，对于某个产业资本来说，它所依赖的生产资料基本局限于在它相邻的层级。比如说，对于第一部类第三层的产业资本来说，它依赖第一部类第二、三、四层的产品作为它的生产资料。类似的，第四层产业资本依赖第四、三层的产品作为其生产资料。第一层产业资本（采矿业等最底层，直接加工自然资源的部门）依赖第二、三层的产品……对于第二部类来说，既要依赖第二部类内部的产品，又需要第一部类内部的产品，不过依赖关系是类似的，并且第二部类对第一部类的依赖的层级是稍微低一点，比如第二部类第三层的部门依赖于第一部类第一、二层的部门。这种依赖关系是比较宽泛的，但是，只要在初始化时固定下来就基本不会变化了。因此，在这个依赖的网络结构中，按照其拓扑结构，价格会存在相互的影响关系。正是这种相互的依赖关系，导致价格与价值的偏移可以说是全局性的。

由于各个产业资本的生产周期不同，这样的流通往往不会是一帆风顺的，或者说，必须要有一定量的存货作为流通的缓冲区来避免流通的暂停。这些存货，一方面由商业资本自己预备，另一方面，也会有产业资本遭遇异常所导致的存货。但是，由于周转的速度不同、供需的潮起潮落、资本增殖的上升期和低估期，这样的流通在大量的混乱中的平衡难免崩溃。这也就是第二卷后半部分关于再生产的研究。那么，我们就不得不来考虑，当这样的流通崩溃时，系统是如何处理的，或者说，当仿真系统的资本主义周期性危机到来时，仿真系统要如何模仿资本主义生产的恢复。

不言而喻，如果是供应的不足，那么促进的是生产的加速，这是可以解决的。但是如果是需求的不足（这里还没有讨论货币流通），那么滞销就会让产业资本产品的出售停滞，从而逆着依赖的链条引发连续的周转停滞。首先遭遇危机的是商业资本，它无法卖出它的商品，因此无法收回它的预付货币资本，它的商品也会逐渐损耗。于是，产业资本的产品也就无法被商业资本收走，产品会存货下来，也同样逐渐损耗。产业资本的周转也就遭受严重打击，周转甚至可能停滞。因此，产业资本与商业资本由于无法获得利润，利润率会普遍降低，有的资本还会因此萎缩甚至死亡。生产停滞，比起逐渐被需求吸收的过剩产品，更多的是过剩产品直接被废弃。根据利润率计算的价格会渐渐的下降（这个价格甚至可能会比成本还低），新的市场缺口随之诞生。当过剩产品逐渐被消灭，流通的供需便会逐渐开始恢复。商业资本逐渐开始运转，这往往是由于工人阶级对必要生活资料的基本需求拉动的。产业资本第二部类会逐渐恢复，随之带动第一部类的恢复。关于这一危机的更加详细的细节，参考资本论第二卷和第三卷关于危机的描述。

## 供需

此处的供需分为两个阶段：产业资本与商业资本之间的供需，商业资本与市场买家之间的供需。我们先来说前者。

### 产业资本与商业资本的供需

产业资本会将其产品按照其计算的生产价格售卖给商业资本，商业资本使用其预付货币资本买下这些产品，后续售卖。关于可能的供给过剩，商业资本会尝试收走所有产业资本的产品，然后在市场上同一售卖。可是，如果出现供给过剩的情况，也就是商业资本无法收走全部商品，只能收走一个部门中的部分产品，那么过剩的产品就不得不滞留在产业资本手中，形成存货。这个存货的存在本身是合理的，用于应对产业资本供给不足的情况。但是，如果这个存货过多，产业资本本身并不具备像商业资本那种优良的保存能力，过剩的存货就会快速损耗。这些滞留在产业资本手中的过剩产品会直接影响产业资本的周转，导致无法收回利润从而利润率下降，甚至资本周转停滞，从而带动平均利润率下降。产业资本供给的过剩往往预示着商业周转的停滞——危机的到来。此时，产业资本为了尽可能保存利润甚至收回成本，会适当降低生产价格，甚至降到生产成本水平。（至于会不会比成本更低，此处保留）

关于可能的需求过剩，产业资本所供给的产品无法满足商业资本在商品市场上的需求，产业资本生产不足。此时，一方面，产业资本可能适当提高生产价格，这也就顺势导致商业价格的提高和最终平均利润率的提高。不过这种情况出现的会比较少，除非是严重供给不足。更重要的是另一方面，产业资本首先是利用它的存货来作为缓冲，然后是快速提高产量。此时要在短时间内爆发产量的方法往往只有一个：提高劳动强度和劳动时间，也就是加强剥削强度。这会在短时间内强行提高产量。渐渐地，产业资本肯定会需要购买更多的劳动资料或生产资料，并在利润率上涨，利润足够的情况下尝试购买更高生产力的生产资料以提高生产力（资本增殖，提高资本有机构成）从而提高产量。这种爆发常见于第一部类产业资本，第一部类产业资本的增值速度也比第二部类资本的速度快的多。至于原因参看第二卷的社会再生产部分。总之，面对商业资本的热烈需求，产业资本的生产动力会大大提高，资本主义社会生产的发展也往往在此时进入狂热期。利润率普遍上涨，商业资本的商品也会被众多不断增殖的产业资本所吸收，直至劳动的过剩致使消费的再次停滞，生产过剩的危机再次到来。

### 商业资本与市场买家的供需

接下来说商业资本与市场买家之间的供需。商业资本所面对的市场买家包括产业资本和所有工人群体（劳动力市场）。产业资本从市场上拿走的，一部分是生产资料，另一部分是劳动资料（都是第一部类的产品）。这里比较麻烦的是生产资料部分中的固定资本（比如机器、乃至厂房）的售卖。我们之前提到，根据第二卷的社会再生产的论述，产业资本（无论第一部类还是第二部类），对固定资本的购置都存在一定的周期，并且这些周期绝非是同步的。因此，商业资本在出售这些固定资本时就面临一个问题：及其不稳定的需求，时而卖不出去，时而供不应求。于是，商业资本为了不亏本，对于这一类的产品的购置就会又一次两极分化：经济发展平稳时期，尽可能少买，尽量保证卖的出去；经济发展狂热时期，又会大量购买，不断地甩卖。不过，总的来说，对于这一类的产品的购置，在平时会在最初购置时就存在一定的抑制。

那么，产业资本从商品市场上购买这些商品时又会如何呢？我们之前提到，产业资本在它的货币资本、生产资本、商品资本三个部分都会留下部分冗余以作其他用处。产业资本在购置劳动资料时，这种冗余最为明显。产业资本会一次购买成批大量的原料，以应对各种生产情况。因此，商业资本与产业资本之间的交易并非是持续性的，而是周期性的。这个周期一般取决于产业资本的生产周期。产业资本在每一个生产周期（一般是）开始，购置一批劳动资料，正如它在生产周期开头，一次付清对劳动力的雇佣一样。商业资本要应对这种周期性的交易，就必须要有足够的存货来应对，也就是说，至少要有满足产业资本一个周期所需劳动资料的量。反过来，商业资本从产业资本那里收购劳动资料也是按照产业资本生活周期进行的。有了这些存货的缓冲，就可以基本保证供需的基本平稳。但是，这种平稳依旧是不稳定的。当商业资本手上的劳动资料供应过剩（此时一般代表着危机时期，生产停滞），商业资本保存这些劳动资料就会不断消耗单纯的流通费用，或者直接废弃过剩的存货。显然的，商业资本的利润率会下降，价格也会下降。直到危机结束，存货（如果还有的话）恢复售出。当供给不足，商业资本自然会将这种需求传递给产业资本，这就回到了上一小节的内容。

产业资本向商业资本购置固定资本时，这里存在两个问题。一方面，产业资本购置这种价格较高的固定资本时，必须需要一定的货币储备，这是它将利润积累特地保存下来的一部分。此外，产业资本要将它的固定资本“用坏”，这和它的生产情况是息息相关的，而商业资本供应这种固定资本，只和它的采购情况相关，而且我们提到，这种采购一般是抑制的。因此，这里的供需要么是过剩的，要么是不足的，平衡的情况几乎是偶然的。这里的情况比较复杂，分两个时期来分别考虑。如果是经济发展的平稳期，商业资本对产业资本的供给一般不足，但是产业资本又非要不可，因此，除了商业资本的存货以外，反而是产业资本必须要提前一段时间预定购买，以免导致生产停滞。如果是经济发展的狂热期，商业资本的供给首先是不足，那么商业资本就会开始加大购置（也就是根据行情解除抑制），存货同样也会加大，来满足繁杂的需求（虽然这几乎是不太可能的，根据第二卷社会再生产部分），然后供给会随着危机的到来而过剩。这些滞留在商业资本手中的固定资本会带来的问题比较大，第一，由于其高昂的价格，如果存货太多，毫无疑问会是沉重的沉默成本，这会严重影响商业资本的复苏；第二，固定资本相较于流动资本（劳动资料），它的损耗（这里不是指量）会很严重，跌价会很快。当危机逐渐过去，生产开始复苏，剩下的存货便会开始发挥作用，商业资本对固定资本的购置再次抑制。

最后，我们来说商业资本与工人群体之间，关于消费资料的供需。毫无疑问，商业资本面临的供需情况和工人的工作情况从而是产业资本的生产情况息息相关。当生产平稳时期，商业资本面临的供需情况会是比较平稳的。重点在于，伴随着劳动的游离（过剩问题）或是劳动的破坏，商业资本的供给会出现过剩的趋势。这种过剩只有在危机时期会比较严重，需求处于持续的低迷，只有价格降低能够缓解。消费资料的存货类似于劳动资料的存货，直到危机恢复，再次售出。消费资料的价格一般会保持平稳，只有在危机时期会降低（其实不会降低多少，甚至会升高，后续关于金融市场的金融危机会提到），或者在高涨时期升高（这个升高可能会比较高）。
