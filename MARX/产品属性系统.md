# 产品属性系统

这里不是直接定义各种各样的商品，而是给出一个商品属性的基本模板。商品大致分为三类：生产资料、劳动资料、消费资料。后续也是根据这三类商品给出多个成员的定义问题（实际上这也就是同时对产业资本进行具体设计）。接下来也是按照这三类别分别说明。

## 量与单价

生产资料（比如大型机器）的量更多说的是它的价格以及**使用寿命**。毫无疑问，生产资料的价格会是相当高的，这表现在再生产它所需的各项材料以及耗费的劳动量的庞大。生产资料的使用寿命具体为，它可以用来参与生产多少产品。或者更具体一点，它**可以使用参与多长的劳动时间（在一个劳动强度范围内）**。比如说，限定使用寿命为1000小时，在劳动强度处于0.8与1.2之间，每参与劳动时间1小时，使用寿命减少1小时。在劳动强度处于1.2与1.5之间，使用寿命减少1.2小时……这个使用寿命是有一定弹性的。值得注意的是，根据第一卷关于机器的论述，**这个机器所能够节约的劳动应该比它的价值更多，甚至多得多**。因此，这个使用寿命的设定绝非随意的。如果其价值（价格）为10000，那么它能够节约的劳动就必须比10000多，因此，在它全部的使用寿命（比如说1000）中，如果按正常使用条件（也就是劳动时间=使用寿命减少）计算，它能够在相同活劳动量推动下，生产出来的比不使用这个机器所生产出来的更多的产品，它们所能够带来的价值，必须多于10000。原本1000个小时劳动能够生产出来的产品总价值只有10000，现在利用机器所生产出来的产品总价值必须多于20000。如果它做不到，产业资本就没什么理由购买它。

劳动资料（比如各种原料）的量更多说的是它的单价以及使用方面的量。就其单价而言，会比较低，毕竟劳动资料往往是成批购置。我们**定义抽象的劳动资料的量就是“物质量”，单位为1**。比如100的劳动资料1（比如200码麻布），20的劳动资料2（比如2斤汽油）……这里不考虑具体劳动资料的单位量，而是使用同一的抽象单位量进行衡量。后续劳动资料与劳动的结合形成产品都是如此计算。因此**单价就是用一批劳动资料的总价格除它的物质量**。比如100的劳动资料1的总价格为200，那么它的单价就是200/100=2。

消费资料（比如食物、衣物等）类似于劳动资料的定义，采用抽象的“物质量”作为量。不过消费资料的购买不是像劳动资料的购买是成批的，而是一小份一小份买的。所以消费资料的物质量的尺度需要设计的细一点，以免误差影响严重。

## 使用价值

生产资料的使用价值首先就是提高技术生产力，**让同一劳动量可以推动更多劳动资料**，提高产量。反过来说，就是同一产量的劳动量的节省。从使用价值层面来看，它本身并不在物质上加入产品，只是优化产品生产的过程。从价值层面来看，生产资料伴随着其在生产中的参与使用，随着它自身的逐渐损耗，它将自己的价值逐渐转移到产品的价值之中。生产资料的使用价值会给生产过程中的劳动量乘上一个**“技术生产力”系数**，比如说2。因此，原本同一个劳动量现在就可以结合原来2倍的劳动资料，从而生产出原来2倍的产品。一个生产周期的产量提升，体现为生产速度的提升。同一个产量所需的劳动量的减少，体现为所需雇佣的减少，从而是成本的减少。对于同一个资本额来说，机器（生产资料）的运用让劳动的需要减少，因而固然会改变资本的有机构成。其次，还是以机器为例，机器的运用可以**降低损耗**并**提高产品的质量**。关于这两个方面，我们放到下文劳动资料部分。

劳动资料的使用价值主要是形成产品，在物质上全部加入形成最终的产品。就像一个化学方程式一样。但是，劳动资料的品质，从而也是其生产层面上的一种使用价值就是：尽可能降低耗损、提高产品质量。关于降低耗损，劳动资料加入产品固然是存在一定耗损的，这个耗损可以根据劳动力的提升以及生产资料的运用而降低，也可以根据劳动资料本身的品质而降低。一定量的劳动资料，先减去其相应的耗损，然后再加入产品。这里也就存在一个**“损耗”**系数。比如，100的劳动资料的最终耗损系数为0.02。那么，实际加入产品形成的就只有$100-100 \times 0.02=98$。第二方面，关于提高产品质量问题。产品的质量，具体来说也是产品的使用价值属性，**对于生产资料而言就是其使用寿命、技术生产力系数；对于劳动资料而言就是其损耗系数；对于消费资料而言就是劳动力恢复系数**。在这里，劳动资料本身的使用价值“提高产品质量”也就表现为对最终的产品的质量的一系列系数乘上一个“提升”指数。

消费资料的使用价值主要是恢复劳动力。劳动力在生产过程中的运用使得劳动力在健康度上存在损耗。运用相应得到的工资购买消费资料从而恢复损耗的劳动力，也就具体表现为健康度的恢复。**消费资料的量和质量（同上文，就是“劳动力恢复系数”）同时决定了它恢复健康度的多少**。比如说100消费资料，劳动力恢复系数为0.4，那么健康度就可以恢复$100 \times0.4=40$。工人购买消费资料的量仅取决于它用于购买的工资的量，但是，最终决定它购买来的消费资料到底能不能完全恢复它的劳动力还要看消费资料的质量。毫无疑问，如果质量很差，那么哪怕买到了足够的消费资料，劳动力还是不能有效恢复；反之，如果质量很好，劳动力就可以得到充足的恢复，甚至提高劳动力的质量。不过后一方面我们忽略，我们也不去考虑产业资本为了盈利偷工减料的问题。此外，对于消费资料的生产使用需要注意一点。对于第二部类产业资本的生产而言，比如说，使用鸡蛋来做蛋糕的产业资本的生产过程，这里的消费资料（鸡蛋）实际上作为劳动资料起作用。于是，它又需要像劳动资料那样计算损耗和质量提升。这里的处理是这样的：当第二部类产业资本在购买第二部类的产品时，它是将其作为劳动资料，而不是消费资料购买的，对于他来说，它购买的产品只表现出劳动资料的性质，而不表现出消费资料的性质，尽管它是第二部类的产品而不是第一部类的产品。

## 损耗

这里关乎两个方面，一是在生产过程中的生产损耗，二是在流通过程中的流通损耗。这里必须强调，生产损耗属于生产技术的属性，同一种产品在不同产业资本的不同生产过程中，生产损耗的系数是不一样的。而流通损耗属于流通技术和产品本身两方面属性。

生产资料的生产损耗在上文中已经提到过了，即伴随着生产资料参与生产过程出现的生产资料使用寿命的减少。这里补充说明一点，我们这里不去考虑生产资料不在生产过程中的损耗，也就是说，对于一天24小时中，生产时间之外所产生的损耗（比如生锈），我们不去考虑它的影响。毕竟这一部分的影响是很小的。至于生产资料的流通损耗。这一方面应该是很小的。比如说，生产资料在流通过程中停滞了比较长的一段时间（假设1000小时），那么对于其使用寿命的减少会比较小，可能只有10小时。也就是说，生产资料的流通损耗的计算方式为：$使用寿命流通损耗=流通时间 \times 生产资料流通损耗系数（很小）$。当然，我们提到过，生产资料可能停滞在产业资本手上，也可能停滞在商业资本手上。两者的流通条件不一样，因此两种情况下的流通损耗系数也不一样。

劳动资料的生产损耗同样在上文提到，即由于技术层面上的问题所带来的物质层面上的损耗，减去损耗的劳动资料，剩下的加入最终产品形成。类似的，我们也不考虑工厂下班时期劳动资料会损耗多少。对这一部分的计算最终落实到“最终损耗系数”的计算。比如，劳动资料本身的损耗技术为0.1，生产资料的运用的影响因子为0.5，劳动力本身的劳动技术力（劳动力的质量）的影响因子为1.0，于是最终损耗系数就是$0.1 \times 0.5 \times 1.0=0.05$。关于流通损耗，也是假设这一方面的影响比较小（虽然不如生产资料）。比如说100劳动资料，在流通过程中停滞1000小时，那么最终量的减少不会超过10。因此，劳动资料的流通损耗计算方式为：$劳动资料物质量损耗（此处取整）=流通时间 \times 劳动资料流通损耗系数（较小）$。

消费资料的生产损耗，实际上也就是上述提到的，当消费资料被作为劳动资料使用时，这里的设计和上述劳动资料的生产损耗设计没有任何不同。重点来说流通损耗。消费资料的流通损耗是比较严重的。比如说，100量的消费资料，只需要100小时就可能全部损耗完。其次，它不仅在量上会损耗，它在质量上也可能产生损耗。我们这里只考虑“劳动力恢复系数”上的损耗。原本“劳动力恢复系数”为1.0的消费资料，经过100小时的流通，其系数就可能降低到0.9。因此，消费资料的流通损耗的计算方式为：$消费资料物质量损耗（此处取整）=流通时间 \times 消费资料量流通损耗系数（较大），消费资料质量损耗=流通时间 \times 消费资料质量损耗系数（较小）$。

## 产品生产

首先，我们假设每种产业资本只有一种产品，但是他们运用的生产资料（不会很多种）、劳动资料可能有多种。我们还是不具体制定各种产业资本之间的产品关系如何，只是给出基本的产品生产关系式模板。为了简便，我们将生产资料简写为“p”，劳动资料简写为“l”，消费资料简写为“c”，劳动量简写为“L”。

比如，$\ce{x l1 + y l2 + z l3 ->[wL][p] n c}$，表示：w劳动量，在生产资料p的参与下，推动x劳动资料1、y劳动资料2、z劳动资料3，最终得到产品——n消费资料。这种设计参考了化学方程式的结构，因此，一方面，这种关系式决定了不同产品之间的物质组成关系，更重要的是，定义了不同产品之间物质量的相互关系。于是，接下来的难点就是：我们要像写一个化学式那样来写一个产品，像配平一个化学方程式那样来配平我们的生产关系式。这里假设两点：首先，产品生产的关系式是始终不变得，换言之，就算有了各种技术上的升级，生产同一种类的产品所需要的资料的种类以及相应配比都是不变的，只会是产品质量的提升和损耗的降低。

关于生产资料的生产，第一部类产业资本在生产这一类产品时，具有两个特点：第一，所需资料量非常大；第二，所需劳动量也很大，但相对于资料量很小（也就意味着资本的有机构成很大）。因此，其基本的生产关系式为：$\ce{x l1 + y l2 + z l3 ->[wL][p]  p1}$。接下来，就是产品的量（使用寿命）的设计。之前提到，资本购置生产资料所耗费的资本必须多于节约的劳动，否则购买它就是无益的。但是，生产的结果（使用寿命）和流通的结果（最终价格）没有理由就是必然一致的，只是由于这样的一种要求，两者存在着间接的协调。这里作这样一个设计。根据生产出这种生产资料的成本价格，去对比劳动力的价格，从而推算这种生产资料在成本上相当于多少劳动力（此处取社会平均劳动），最后乘$12 \times 360$。比如，生产的成本价格为10000，当前的劳动力价格为100，故相当于100个劳动力的价格，最后乘$12 \times 360$，得到使用寿命应为$100 \times 12 \times 360=432000小时$。当然，这个使用寿命只是基本的使用寿命，根据上文关于“提高产品质量”的描述，可以乘上一个提升的系数。至于技术生产力系数的设计，这个属于基本的质量属性，简单设定基础值为2，可以不断随着生产技术提升而提升。

关于劳动资料的生产，第一部类产业资本再生产这一类产品时，具有两个特点：所需资料量较少，所需劳动量较少。因此，其基本的生产关系式为：$\ce{x l1 + y l2 + z l3 ->[wL][p]  nl}$。产品劳动资料的量的设计，更多关注所使用原料的量。比如，如果产品为棉布，那么这个棉布的量更多地取决于其原料——棉花的量。同时，产品的量也受原料量的限制，100斤棉花无论如何也产不出200斤棉布。因此，可以简单设计为，最终产品的量为原料量的加权求和。比如上述的n，可以设计为：$n=0.8x + 0.5y +0.7z$。权重代表了原料在多大程度上加入产品，权重不会超过1。

关于消费资料的生产，在一定程度上与劳动资料的生产相似。基本的生产关系式为：$\ce{x l1+y l2 + z c1 + m c2 ->[wL][p]n c3}$。无论最终产品消费资料$c_3$最终到底是作为消费资料用于个人消费还是作为劳动资料用于生产消费，产品量的定义都是一致的。但是当作为消费资料时，还必须考虑另一个量——劳动力恢复系数。生产1单位的消费资料，恢复劳动力健康度刚好等于其劳动力恢复系数，而这个系数应该至少比生产它所耗费的劳动多。我们简单假设基础值为1，可以不断提升。

## 总结

|              | 生产资料                                   | 劳动资料                             | 消费资料                     |
| ------------ | ------------------------------------------ | ------------------------------------ | ---------------------------- |
| 量定义       | 使用寿命                                   | 物质量                               | 物质量                       |
| 使用价值     | 技术生产力<br />降低损耗<br />提高产品质量 | 产品形成<br />损耗<br />提高产品质量 | 劳动力恢复<br />作为劳动资料 |
| （流通）损耗 | 很小                                       | 较小                                 | 较大                         |
| 生产量关系   | 推算节省劳动                               | 原料物质量加权                       | 原料物质量加权               |
| 属性提升空间 | 使用寿命<br />技术生产力系数               | 损耗系数                             | 劳动力恢复系数               |
