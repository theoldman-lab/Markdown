# 《资本论》第三卷 第五篇"利润分为利息和企业主收入。生息资本"

## 一、篇目定位

第五篇在《资本论》第三卷中的位置是独特的——它既是"利润"范畴分解的最后环节（产业利润→平均利润→利息+企业主收入），又是全书中最系统论述**信用制度**的篇章。全篇共16章（第21—36章），可划分为四大理论板块，呈现出一条清晰的"从抽象上升到具体"的逻辑线索：

> **生息资本的一般规定（21—24） → 信用制度的理论分析（25—28） → 银行资本与货币资本/现实资本关系（29—32） → 信用制度的具体运行、国际关系与历史前史（33—36）**

## 三、贯穿全篇的核心理论线索

### 1. 利润分解的三重逻辑

```
产业利润（特殊）→ 平均利润（产业+商业）→ 利息 + 企业主收入
```

- 第21—23章完成了利润形式从"平均利润"到"利息+企业主收入"的分解
- 这一分解的理论必要性：只有在资本所有权与资本职能分离的条件下，利润的内部分割才成为可能
- 第23章的核心发现：**单纯的量的分割一旦发生在不同的人之间，就硬化并为质的分割**——利息来自资本所有权，企业主收入来自资本职能

### 2. 虚拟资本的生成谱系

全篇识别了三种虚拟资本的生成机制：

| 形式 | 生成机制 | 本质 |
|------|----------|------|
| **国债券** | 国家借款已耗费→债券代表对税收的索取权 | 最纯粹的虚拟资本——负数表现为资本 |
| **股票** | 所有权证书的市场价值与实际资本同时存在 | "双重存在"——市场价值独立于现实资本 |
| **银行存款** | 同一货币反复存储→乘数效应→存款倍增 | 同一货币承载数倍"货币资本" |

核心公式——**"资本化"**：年收入 ÷ 利息率 = 幻想的资本价值。一切有规则的货币收入都被倒算为生息资本，由此"和资本的现实增殖过程的一切联系就彻底消灭干净了"。

### 3. 借贷资本与产业资本的运动对立

这是第30章的核心发现，贯穿第31—32章：

| 周期阶段 | 利息率 | 产业资本 | 方向 |
|----------|--------|----------|------|
| 危机后停滞 | 极低 | 收缩 | **相反** |
| 复苏 | 低 | 扩大 | 一致 |
| 繁荣顶点 | 平均水平 | 最大 | 一致 |
| 危机 | 极高 | **过剩** | **相反** |

**"表现在利息率上的借贷资本的运动，和产业资本的运动，总的说来是方向相反的。"**

这一规律瓦解了一切将利息率视为经济健康指标的肤浅判断：低利息率可能是产业萎缩（而非繁荣）的标志。

### 4. "信用主义→货币主义"的周期性转化

这是全篇关于危机本质的核心命题：

**繁荣期**：信用排挤货币，票据大量替代现金，一切都在信用的基础上运行

**危机期**：信用突然崩溃→"信用主义突然转变成货币主义"→每个人都只要现金→昨天充足的银行券"一夜之间消失"→"为了几百万货币，必须牺牲许多百万商品"

**哲学命题**："货币主义本质上是天主教的；信用主义本质上是基督教的……但是，正如基督教没有从天主教的基础上解放出来一样，信用主义也没有从货币主义的基础上解放出来。"（第35章）

信用永远无法摆脱对货币金属基础的依赖——每一次危机都是这一被压抑的基础的周期性复仇。

### 5. "资本主义范围内的扬弃"——信用制度的过渡功能

第27章与第36章首尾呼应，构成了全篇最具历史哲学意义的论证：

- 股份公司＝**消极的扬弃**：私人资本→社会资本的形式，但劳资对立未变
- 合作工厂＝**积极的扬弃**：工人在旧形式内打开的第一个缺口，劳资对立已被扬弃
- 信用制度＝**过渡杠杆**："在由资本主义的生产方式向联合起来劳动的生产方式过渡时，信用制度会作为有力的杠杆发生作用；但是，它仅仅是和生产方式本身的其他重大的有机变革相联系的一个要素。"

这就既肯定了信用的历史进步作用（为社会主义准备物质条件），又批判了圣西门主义者和蒲鲁东主义者将信用本身视为解放手段的幻想。

### 6. 货币与资本的区分——全篇最基础的理论前提

如果没有这一区分，全篇的分析框架就会崩溃。马克思（和恩格斯）区分了：

| | 货币 | 资本 |
|--|------|------|
| **银行信贷** | 抵押贷款、票据贴现（支付手段） | 个人信用贷款（追加资本） |
| **流通领域** | 收入的货币形式（消费者→零售商，购买手段） | 资本的货币形式（资本家间，支付手段） |
| **危机需求** | 对支付手段的需求 | 对追加资本的需求（极少，商品资本已过剩） |
| **金流出** | 货币问题（国际支付手段） | 同时也是资本问题（输出的是实际价值） |

**"通货原理"派的根本错误**：将一切货币需求都等同于资本需求→以此论证高利息率的合理性→为银行家利益辩护。

**"银行学派"的不足**：虽然指出了货币≠资本，但未能严格区分，且站在银行家立场上混淆了通货与资本。

## 四、方法论特征

1. **从抽象上升到具体**：生息资本的一般规定（21—24）→信用制度（25—28）→银行资本解剖（29）→货币资本/现实资本动态（30—32）→具体运行机制（33—35）→历史回溯（36）

2. **理论与经验的循环互证**：每一个理论命题都通过1847年/1857年危机的议会证词获得经验规定；每一条经验材料都被理论范畴组织进分析框架

3. **双重批判**：同时批判"通货原理"派（第26、34章）和"银行学派"（第28章），在两派之间建立独立的唯物主义立场

4. **历史与逻辑的统一**：从"洪水期前"的高利贷资本（第36章）到现代信用制度再到未来联合生产的过渡——逻辑的展开同时是历史的重演

5. **阶级分析的贯彻**：全篇始终揭示信用制度背后的阶级利益——货币资本家 vs. 产业资本家 vs. 工人阶级。"这伙匪帮既不懂生产，又同生产没有关系"（第33章）；1844年银行法是"周期地把产业利润放到高利贷者的钱袋中去"的"巧妙的安排"（第34章）

## 五、全篇的理论贡献总览

1. **完成利润形式的全部分解**：产业利润→平均利润（产业+商业）→利息+企业主收入→完成了资本主义收入形式体系的建构

2. **创立虚拟资本理论**：国债券（纯粹虚拟）、股票（双重存在）、存款（乘数）——提供了分析一切金融现象的元框架

3. **建立借贷资本/产业资本运动对立规律**：揭示了利息率不能作为经济状况指标的根本原因

4. **完成资本拜物教的批判**：G—G′ 是拜物教的完成形态；"生息资本一般是一切颠倒错乱形式之母"

5. **提出信用制度"过渡功能"的辩证命题**：信用既是危机和欺诈的最有效工具，又是通向新生产方式的过渡杠杆

6. **摧毁资本主义的最终伦理幻象**："不仅利润来自对他人劳动的占有，而且用来推动和剥削他人劳动的资本也来自他人的财产"

7. **阐明货币危机的内在必然性**：货币危机不是"管理不善"，而是资本主义生产方式本身的产物——商品中的社会劳动必须以外在的货币来表现

8. **完成生息资本的完整历史谱系**：高利贷资本（前资本主义）→从属于产业资本的信用制度→向联合生产的过渡
