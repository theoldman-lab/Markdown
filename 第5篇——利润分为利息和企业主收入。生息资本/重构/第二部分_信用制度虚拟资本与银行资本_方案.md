# 第二部分重构方案——信用制度、虚拟资本与银行资本

---

## 定位

第21—24章完成生息资本基本理论的介绍后，进入信用制度领域。本部分作为**初步介绍**——建立商业信用、虚拟资本、银行制度的基本范畴，不涉及与各学派的论战（留待第四部分），不涉及危机事件叙述（已独立为第三部分），不涉及信用四大作用的全面总结（留待第五部分）。

---

**来源**：第25章第3—9行（马克思正文） + 第33—35行（图克论信用） + 第37—39行（科克兰论信用）

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第3—9行 | 票据＝定期支付的凭证、票据作为商业货币流通、票据是信用货币（银行券）的基础——真正的信用货币不是以货币流通为基础，而是以票据流通为基础 |
| 第25章第33—35行 | 图克：信用的最简定义——"一种适当的或不适当的信任，使一个人把一定的资本额委托给另一个人" |
| 第25章第37—39行 | 科克兰：信用在产业关系内部的链条——"每一个人都是一只手借入，另一只手贷出" |

### 第二节：虚拟资本的形成

**来源**：第25章第13—31行（利瑟姆、博赞克特数据） + 第29章第15—37行（资本化机制）

#### 2.1 票据流通的规模与融通票据

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第13—24行 | 利瑟姆：1839年全英票据总额5.28亿镑、流通额1.32亿镑；"汇票是通货最大组成部分"；1.53亿镑债务 vs. 1,400万镑金；**"融通票据，就是人们在一张流通的汇票到期以前又签发另一张代替它的汇票，这样，通过单纯流通手段的制造，就创造出虚拟资本"** |
| 第25章第25—31行 | 博赞克特：票据交换所每天平均支付300万镑以上，需用货币仅20万镑；汇票通过背书，每天完成1,800万镑货币的职能 |

#### 2.2 资本化——虚拟资本的生成机制

| 提取段落 | 内容要点 |
|----------|----------|
| 第29章第15—17行 | 生息资本的形式造成：每个有规则的货币收入都表现为一个资本的利息→"货币收入首先转化为利息，有了利息，然后得出产生这个货币收入的资本" |
| 第29章第18—27行 | 国债：资本已由国家花掉→债权人只有一张索取年税收的权利证书→**"国债的资本仍然是纯粹的虚拟资本；一旦债券不能卖出，这个资本的假象就会消失"**。"在国债的场合，负数表现为资本" |
| 第29章第29—35行 | 劳动力被"资本化"——工资被看作利息，劳动力被看作提供利息的资本。资本家思考方式的错乱达到顶点。"不幸有两件事情：第一，工人必须劳动才能获得这种利息；第二，他不能通过转让把劳动力的资本价值转化为货币" |
| 第29章第37行 | **"人们把虚拟资本的形成叫作资本化。人们把每一个有规则的会反复取得的收入按平均利息率来计算…这样就把这个收入资本化了。因此，和资本的现实增殖过程的一切联系就彻底消灭干净了。"** |

### 第三节：银行制度

**来源**：第25章第43—59行（银行起源与结构） + 第63—65行（图克论银行业务两分） + 第81—113行（吉尔巴特论银行资本与业务）

#### 3.1 银行的起源与基本结构

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第43—45行 | 从货币经营业到银行业——实业家的准备金保管、货币出纳、国际支付和金银贸易集中在货币经营者手中→"货币的借入和贷出成了他们的特殊业务…他们以货币资本的实际贷出者和借入者之间的中介人的身份出现"。**"银行家成了货币资本的总管理人。"** |
| 第25章第45行 | 银行的利润：**"借入时的利息率低于贷出时的利息率"** |

#### 3.2 银行借贷资本的四个来源

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第49行 | **第一**：产业资本家的准备金→集中到银行→"作为共同的准备金集中起来，就可以限制到必要的最低限度" |
| 第25章第51行 | **第二**：货币资本家的存款 |
| 第25章第53行 | **第三**：一切阶级的货币积蓄——"小的金额是不能单独作为货币资本发挥作用的，但它们结合成为巨额，就形成一个货币力量" |
| 第25章第55行 | **第四**：各种只是逐渐花费的收入 |

#### 3.3 银行信贷的形式

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第57行 | 贷放形式：票据贴现、以个人信用为基础的直接贷款、有价证券/国债券/股票/提单/栈单抵押贷款、存款透支 |
| 第25章第59行 | 银行家提供的信用形式：向其他银行签发汇票/支票、开立信用账户、发行本行银行券。**"银行券无非是向银行家签发的、持票人随时可以兑现的、由银行家用来代替私人汇票的一种汇票"**。"对银行来说具有最重要意义的始终是存款" |

#### 3.4 吉尔巴特论银行资本

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第83行 | 银行资本＝投资＋借入的银行资本。借入资本通过三条途径取得：接受存款、发行银行券、签发汇票 |
| 第25章第91行 | **"银行家是把别人的货币贷给他的顾客"** |
| 第25章第97—99行 | 存款银行用转账的办法节约流通手段→"靠转账的办法，存款制度能够达到完全不使用金属货币的程度" |
| 第25章第107行 | **"银行的目的在于便利营业。但一切便利营业的事情，都会便利投机。"** |

### 第四节：存款乘数与虚拟资本的膨胀

**来源**：第25章第115—117行（《通货论》论存款乘数） + 第29章第57—77行（存款机制与斯密论述）

| 提取段落 | 内容要点 |
|----------|----------|
| 第25章第115—117行 | **"我今天存在A处的1,000镑，明天会被支付出来，形成B处的存款…这1,000镑货币，通过一系列的转移，可以成倍地增长为一个绝对无法确定的存款总额。"** 苏格兰：货币流通额不超过300万镑，存款却有2,700万镑 |
| 第29章第63行 | 存款的双重作用：作为生息资本贷出（不留在保险柜里）＋作为账面项目互相抵消 |
| 第29章第65行 | **"随着生息资本和信用制度的发展，一切资本好像都会增加一倍，有时甚至增加两倍…这种'货币资本'的最大部分纯粹是虚拟的。"** |
| 第29章第67—74行 | 斯密论货币借贷：同一些货币可以连续进行许多次借贷和许多次购买——A→W→B→X→C→Y→D，"同一些金币或纸币，在数日之内，就可以用来借贷三次和购买三次" |
| 第29章第75—77行 | 斯密关于借贷的话也适用于存款——"同一些货币可以充当不知多少次存款的工具"。苏格兰实例：货币流通额不超过300万镑，存款却有2,700万镑 |

### 第五节：银行资本的组成部分——虚拟资本的类型学

**来源**：第29章第3—11行（银行资本二分） + 第39—51行（股票的双重存在） + 第43—47行（证券独立运动与危机） + 第49—51行（总结——索取权证书）

#### 5.1 银行资本的二分结构

| 提取段落 | 内容要点 |
|----------|----------|
| 第29章第3—11行 | **银行资本＝现金（金/银行券）＋有价证券**。有价证券再分为：商业证券（汇票）＋公共有价证券（国债券、国库券、股票、地产抵押单）。按来源分为：银行家自有投资＋别人的存款。**"不论银行家只用自有的资本来经营业务，还是只用在他那里存入的资本来经营业务，银行家资本的上述区分仍然不变"** |

#### 5.2 国债券——最纯粹的虚拟资本

已在第二节"资本化"中详述，此处可标注"见第二节第2.2小节"。

#### 5.3 股票——现实资本的"双重存在"

| 提取段落 | 内容要点 |
|----------|----------|
| 第29章第39行 | 铁路、矿山、轮船等公司的股票代表现实资本。**"但是，这个资本不能有双重存在：一次是作为所有权证书即股票的资本价值，另一次是作为在这些企业中实际已经投入或将要投入的资本。它只存在于后一种形式"** |
| 第29章第41行 | 所有权证书的价值的独立运动——"它们已经成为商品，而这些商品的价格有独特的运动和决定方法" |
| 第29章第41—42行 | 股票市场价值＝预期年收益÷利息率。企业提供10%而非5%→利息率5%时，100镑名义股票→市场价值200镑。"**它的价值始终只是资本化的收益**" |
| 第29章第43行 | "这种证券的市场价值部分地有投机的性质，因为它不是由现实的收入决定的，而是由预期得到的、预先计算的收入决定的" |
| 第29章第45行 | 危机中双重跌落：利息率提高 + 大量抛售以变现→**"它们在危机中的贬值，会作为货币财产集中的一个有力的手段来发生作用"** |
| 第29章第47行 | **"只要这种证券的贬值或增值同它们所代表的现实资本的价值变动无关，一国的财富在这种贬值或增值以后，和在此以前是一样的。"**——"一个国家就决不会因为名义货币资本这种肥皂泡的破裂而减少分文" |

#### 5.4 总结

| 提取段落 | 内容要点 |
|----------|----------|
| 第29章第49行 | **"所有这些证券实际上都只是代表已积累的对于未来生产的索取权或权利证书"** |
| 第29章第51行 | "货币资本的积累，大部分不外是对生产的这种索取权的积累，是这种索取权的市场价格即幻想的资本价值的积累" |
| 第29章第59行 | **"银行家资本的最大部分纯粹是虚拟的"**——由债权（汇票）、国债券（代表过去的资本）和股票（对未来收益的支取凭证）构成 |
