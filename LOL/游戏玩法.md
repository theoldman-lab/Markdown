## 1. 版本概述

本玩法从底层重构了英雄联盟的核心系统：**召唤师技能、防御塔规则、野区生态、视野系统与兵线节奏**。所有改动围绕一个统一的主题——**传送点（Waypoint）**，力求让地图上每一个战略节点都具备争夺价值，同时在加速游戏节奏的同时保留决策深度。

### 主要改动一览

| 系统 | 改动方向 |
|------|----------|
| 防御塔 | 新增传送点机制；镀层被破后敌方可短暂占领；高地塔与水晶合并；门牙塔移除、主水晶加强 |
| 野区 | 史诗级野怪刷新顺序与掉落重做；龙坑与大龙坑演化为中心传送点；双史诗怪全死后刷新阵营传送点 |
| 召唤师技能 | **全部移除**，效果迁移至鞋子、药水、饰品与史诗Buff |
| 饰品 | 五大免费饰品，涵盖视野、扫描、远视、真眼（辅助）、打野 |
| 小兵 | 每波+2小兵；对塔伤害提升；新增召唤/强化小兵装备 |
| 装备 | 鞋子、药水、饰品三大系统承接原召唤师技能效果 |

## 2. 核心设计理念

1. **传送点无处不在**：让地图上的每一个据点——防御塔、史诗怪坑、Buff营地——都成为可供争夺/使用的传送点，极大提升地图资源的价值密度。

2. **召唤师技能物品化**：将原本「开局选定、整局固定」的召唤师技能拆解到装备/消耗品/饰品中，使得玩家可以根据对局形势灵活切换战术。

3. **节奏驱动**：通过固定的史诗野怪刷新时间线（15m 龙 → 20m 先锋 → 25m 远古龙 → 30m 男爵），为游戏提供清晰的时间轴，引导双方在每个阶段围绕不同的核心目标展开博弈。

4. **占领与争夺**：中立传送点的「摧毁后可占领」机制将进攻与防守融为一体——攻破敌方据点不仅削弱对手，还能为自己赢得战略机动性。

5. **简化但并非浅化**：高地水晶合并、门牙塔移除、召唤师技能集中管理等设计旨在减少冗余元素，但通过传送点网络、多种饰品博弈、Buff 叠加等新维度，保留了游戏的策略深度。

## 3. 防御塔传送点

### 3.1 传送机制

- 全场所有防御塔均自带一个**传送点**。
- 玩家在己方传送点附近按下 **B 键**，可打开传送点地图，选择另一个**己方已激活的传送点**进行传送。
- 传送引导时间为 **4 秒**，引导期间受到敌方英雄或防御塔伤害则中断。
- 每个传送点拥有独立的 **内置冷却（CD）**，单次传送后该传送点进入 **90 秒** CD 期。期间该传送点不可作为出发地，仅可作为目的地。

### 3.2 镀层与占领

- 防御塔初始拥有 **5 层镀层**，每层镀层提供 **200 点额外生命值**（镀层部分受到伤害时获得 **20% 减伤**）。
- 镀层每被摧毁一层，击破者（最近造成伤害的英雄）及其附近队友获得 **160 金币**（5 人平分时每人约 50 金币）。
- 当防御塔**全部 5 层镀层被敌方摧毁**后：
  - **外塔、二塔**：该防御塔的传送点被**敌方暂时占领**。占领持续 **180 秒**，期间敌方阵营可使用该传送点进行 B 传送。180 秒后传送点自动归还防守方。
  - **高地防御塔**：传送点**无法被敌方占领**，始终保持为防守方所有。
  - 敌方占领期间，防守方仍可通过**普攻摧毁镀层重新夺回**传送点控制权（夺回后该塔恢复 1 层镀层）。
- 镀层在游戏进行到 **14 分钟**后自然脱落（自动消失），此后防御塔生命值回归正常。但传送点的占领规则仍按照「是否被完全摧毁」判定。

### 3.3 高地与水晶

- **高地水晶移除**，其功能合并至**高地防御塔**。
- 高地防御塔被摧毁后：
  - 敌方在该条兵线生成**超级兵**（效果等价于原高地水晶被摧毁）。
  - 该高地防御塔的传送点**永久失效**（双方均不可使用）。
- 高地防御塔**不会自然重生**。

### 3.4 主水晶

- 原主水晶前的**两座门牙防御塔移除**。
- 主水晶新增 **7 层镀层**，每层镀层提供 **400 点额外生命值**。
- 主水晶传送点**无冷却时间**，防守方可在任意时间从主水晶传送至任何己方激活的传送点。但传送出去的英雄在 **60 秒**后才可再次使用主水晶传送。
- 主水晶镀层被破坏后不可恢复，且每破一层镀层，在该条兵线生成一波强化兵线。

## 4. 野区传送点

### 4.1 下半区：龙坑生态链

#### 4.1.1 元素巨龙（15 分钟刷新）

| 属性 | 数值 |
|------|------|
| 生命值 | 8000（+200/分钟） |
| 攻击力 | 180 |
| 护甲/魔抗 | 50 |
| 刷新间隔 | 单条击杀后 5 分钟刷新下一条（共 3 条不同元素） |

**掉落——龙心（Dragon Heart）：**
- 击杀元素巨龙的英雄获得 **1 颗龙心**（物品栏中可堆叠，最多 3 颗）。
- 使用龙心可召唤对应元素的**元素巨龙**，巨龙持续 **90 秒**，自动沿兵线推进并攻击敌方单位。
- 龙心拥有者（及附近 1000 范围内的友方非战斗状态英雄）可**右键骑乘**元素巨龙。骑乘时：
  - 巨龙获得 **无视地形飞行** 能力，移动速度提高 **30%**。
  - 骑乘者可使用方向键控制巨龙飞行方向，持续时间 **8 秒**。
  - 飞行结束或骑乘者主动脱离后，巨龙继续自动推进。
- 元素巨龙周围 **800 范围**内的友方英雄获得对应的**元素龙魂 Buff**：
  - **炼狱亚龙**：攻击力和法术强度 +8%（乘算）。
  - **海洋亚龙**：每秒恢复 2.5% 已损失生命值。
  - **山脉亚龙**：护甲和魔抗 +8%（乘算）。
  - **云端亚龙**：移动速度 +10%，技能急速 +10。
  - （本玩法下元素巨龙类型各有 25% 几率刷新，首条随机。）

#### 4.1.2 远古巨龙（25 分钟刷新）

| 属性 | 数值 |
|------|------|
| 生命值 | 14000（+250/分钟） |
| 攻击力 | 300 |
| 护甲/魔抗 | 80 |

**掉落——远古龙魂 Buff：**
- 全体友方英雄获得，持续 **180 秒**（死亡不丢失）。
- 效果：对生命值低于 20% 的敌方英雄造成的伤害附带**灼烧**，灼烧伤害为 **每秒 100~200（基于等级）真实伤害**，持续 3 秒。若灼烧期间目标死亡，则对周围敌方单位施加相同的灼烧。
- 等价于原**引燃（Ignite）**的斩杀效应，但更偏团战范围。

#### 4.1.3 中立传送点（远古巨龙被击杀后）

- 远古巨龙被击杀后，龙坑内刷新一个**中立传送点**。
- 任意一方可通过**普攻 3 次**摧毁该传送点，摧毁方获得占领权。
- 占领状态持续至**敌方同样通过普攻 3 次摧毁并夺回**。
- 占领方可在龙坑传送点使用 B 键传送至任意己方传送点，或接收来自其他己方传送点的传送。
- 中立传送点无独立冷却时间，但其上的传送计入出发地/目的地防御塔的 CD。

### 4.2 上半区：大龙坑生态链

#### 4.2.1 峡谷先锋（20 分钟刷新）

| 属性 | 数值 |
|------|------|
| 生命值 | 10000（+200/分钟） |
| 攻击力 | 250 |
| 护甲/魔抗 | 60 |

- 击杀后掉落 **先锋之眼**（仅可使用一次，持有 180 秒后消失），效果与原峡谷先锋一致：引导后召唤峡谷先锋沿兵线推进，对防御塔造成大量真实伤害。
- 先锋之眼可主动使用召唤峡谷先锋，无需引导（瞬发），但伤害降低 20%（对比原版）。

#### 4.2.2 纳什男爵（30 分钟刷新）

| 属性 | 数值 |
|------|------|
| 生命值 | 18000（+300/分钟） |
| 攻击力 | 400 |
| 护甲/魔抗 | 100 |

**掉落——虚空男爵 Buff：**
- 全体友方英雄获得，持续 **180 秒**（死亡不丢失）。
- 效果：对敌方英雄造成伤害时，使其造成的伤害降低 **30%**，持续 **3 秒**（每个英雄独立冷却 10 秒）。
- 移除原版男爵 Buff 的小兵增益（强化兵线的功能由小兵装备承担，见 [7.2 节](#72-新装备)）。
- 等价于原**虚弱（Exhaust）**的减伤机制，但为被动触发。

#### 4.2.3 中立传送点（纳什男爵被击杀后）

- 与龙坑传送点机制一致：击杀男爵后在大龙坑刷新中立传送点，摧毁 3 次后占领，敌方可反夺。

### 4.3 阵营传送点（终局机制）

- 当**远古巨龙**和**纳什男爵**均已被击杀后，**5 分钟**后触发终局事件：
  - **上方红 Buff 区**：刷新红色方专属传送点。
  - **下方红 Buff 区**：刷新红色方专用传送点。
  - **上方蓝 Buff 区**：刷新蓝色方专用传送点。
  - **下方蓝 Buff 区**：刷新蓝色方专用传送点。
- 阵营传送点初始为**中立**状态。任意一方可通过**普攻 5 次**摧毁并占领。
- 被占领后敌方同样普攻 5 次可夺回。
- 阵营传送点：只有所属阵营可将其作为**出发地**；作为**目的地**时双方皆可使用（若已被占领）。

### 4.4 传送点汇总

| 类型 | 刷新时间 | 占领条件 | 冷却 |
|------|----------|----------|------|
| 防御塔传送点 | 全局存在 | 镀层全破后被敌方占领 180 秒 | 90 秒/塔 |
| 高地传送点 | 全局存在 | 不可占领 | 90 秒 |
| 主水晶传送点 | 全局存在 | 不可占领 | 无冷却 |
| 龙坑中立传送点 | 远古龙被击杀后 | 普攻 3 次摧毁后占领 | 依赖防御塔 CD |
| 大龙坑中立传送点 | 男爵被击杀后 | 普攻 3 次摧毁后占领 | 依赖防御塔 CD |
| 阵营传送点 | 双史诗被击杀后 5 分钟 | 普攻 5 次摧毁后占领 | 独立 120 秒 CD |

## 5. 召唤师技能

### 5.1 移除与迁移

**所有 9 个召唤师技能全部移除**。其效果按以下方式迁移到装备/物品系统中：

| 原召唤师技能 | 迁移后的载体 | 获取方式 |
|-------------|-------------|----------|
| 幽灵疾步 (Ghost) | 鞋子系主动效果 | 购买对应鞋履 |
| 闪现 (Flash) | 鞋子系主动效果 | 购买对应鞋履 |
| 治疗术 (Heal) | 药水系主动效果 | 购买对应药水 |
| 净化 (Cleanse) | 药水系主动效果 | 购买对应药水 |
| 屏障 (Barrier) | 药水系主动效果 | 购买对应药水 |
| 传送 (Teleport) | 传送点机制 | 全局可用（B 键传送） |
| 引燃 (Ignite) | 远古龙魂 Buff | 击杀远古巨龙 |
| 虚弱 (Exhaust) | 虚空男爵 Buff | 击杀纳什男爵 |
| 惩戒 (Smite) | 紫色饰品主动效果 | 免费获得（打野系） |

### 5.2 设计意图

- **鞋子系列**承担「机动性」职能：玩家根据局势选择短 CD 疾跑（拉扯型）还是长 CD 闪现（爆发/逃生型）。
- **药水系列**承担「生存」职能：消耗品式使用，用完需回城补给。
- **史诗 Buff** 承担「终结技」职能：远古龙魂 = 进攻向斩杀，虚空男爵 = 防御向减伤，二者在团战中的作用对标原先的引燃与虚弱，但效果更强、持续时间有限。
- **传送**彻底融入传送点网络，不再作为独立召唤师技能，使得边路支援的成本与收益通过传送点布局来动态平衡。
- **惩戒**转为打野专属饰品，强化打野位的独特定位。

## 6. 饰品

### 6.1 概述

所有饰品在商店中**免费购买**，开局 30 秒后可首次购买，同类别可随时在商店免费更换（更换后存在 60 秒冷却才可再次更换）。

玩家同时只能装备 **1 个**饰品（占用原饰品槽位）。

### 6.2 饰品详述

#### 金色饰品——远见守卫（假眼）

- **主动效果**：放置一个隐形守卫（假眼），提供周围 900 范围的视野，持续 **90 秒**。
- **充能**：每 **120 秒** 获得 1 层充能，最多储存 **2 层**。
- **生命值**：3 点，仅可被真眼或扫描发现并破坏。
- 与原版监视图腾一致。

#### 红色饰品——神谕透镜（扫描）

- **主动效果**：扫描周围 750 范围内的视野，揭示并沉默隐形守卫和陷阱，持续 **10 秒**。
- **充能**：每 **90 秒** 获得 1 层充能，最多储存 **2 层**。
- 扫描期间若发现敌人，敌方英雄模型会以红色轮廓高亮显示 1 秒。
- 与原版神谕透镜功能一致。

#### 蓝色饰品——远见改造（远视）

- **主动效果**：在地图任意位置放置一个**可见的远视守卫**，提供 500 范围视野，持续 **永久**（直到被摧毁）。
- **生命值**：1 点。
- **射程**：全图。
- **充能**：每 **180 秒** 获得 1 层充能，最多储存 **2 层**。
- 与原版远见改造功能一致。

#### 绿色饰品——真视之眼（辅助系）

- **主动效果**：放置一个**可见的真视守卫**，提供 900 范围视野，**可揭示并禁用范围内的隐形守卫和陷阱**，持续**永久**（直到被摧毁）。
- **生命值**：4 点。
- **充能**：每 **90 秒** 获得 1 层充能，最多储存 **2 层**。
- **附加效果**：装备绿色饰品的玩家在对敌方英雄造成伤害或为友方英雄提供治疗/护盾时，获得 **20 金币**（每 30 秒最多触发 3 次）。此效果在游戏前 20 分钟有效。
- **备注**：原版「真眼（控制守卫）」作为消耗品已被移除。绿色饰品成为唯一的真视来源，同时继承原辅助装备的金币获取机制，免费提供且不占额外格子。

#### 紫色饰品——野区游灵（打野系）

- **主动效果 1——野区游灵**：在目标野区位置释放一个**幽灵游灵**，游灵会在周围 2000 范围内自动巡逻，持续 **120 秒**。游灵提供 300 范围的视野，可跨越地形移动。敌方英雄进入游灵 500 范围时，游灵会发出警报并暴露该英雄位置（小地图头像可见），持续 3 秒。
- **充能**：每 **150 秒** 获得 1 层充能，最多储存 **2 层**。
- **游灵属性**：3 点生命值，仅可被敌方扫描/真眼发现后普攻 3 次摧毁。
- **主动效果 2——惩戒（Smite）**：对目标野怪或史诗野怪造成 **600~1200（基于等级）** 真实伤害。对敌方小兵造成等量伤害（用于清线/反制小兵召唤装备）。
- **惩戒充能**：独立于游灵充能，开局获得 1 层，每 **90 秒** 充能 1 层，最多储存 **2 层**。
- **惩戒升级**：使用 5 次惩戒后，惩戒升级为**挑战惩戒**（对敌方英雄使用时标记目标 4 秒，期间你对目标的伤害提高 10%）或**深寒惩戒**（偷取目标 20% 移动速度，持续 2 秒），在商店中自由切换。

> 💡 **设计说明**：原打野装备（打野刀）被移除。紫色饰品的惩戒效果完全替代原打野刀的刷野、控龙/控先锋、反野的核心功能。同时游灵提供野区控制视野，使打野位的信息能力与刷野能力统一在一个饰品中。
